FontFamilj

font-family är ett element som används inom CSS för att beskriva vilket font som skall användas. Ett font-family element kan även innehålla flera fonts som läses in i fall din webläsare inte skulle stödja huvud fonten som är specifierad. Elementet innehåller två olika "familjer", family-name och generic-family. När man definerar font i CSS börjar man med fonten som önskas och avslutar med den generiska typen så att webläsaren kan välja något liknande till denna om inget annat finns tillgängligt.

Enheter

När man vill sätta storlek på sin text finns det en hel del fördefinerade storlekar att utnyttja, dessa sträcker sig från xx-small ända upp till xx-large. Det finns även möjlighet att sätta storlek utifrån en procent på det initiala värdet genom att använda "#%" där "#" ersätts med din önskade storlek.

Vill man ändra stilen på sin text finns det även där en del fördefinerade val att välja mellan. Man kan t.ex. använda sig av en em-tag för att göra texten kursiv på detta sätt. Andra exempel på detta kan också vara fetstil.


Font-size: small, med röd färg

Font-size: x-large, med blå färg
