• mark - Med den här taggen kan man markera text. Exempel på detta kan vara Viktig text. .
  • wbr - Om man har ett långt ord kan man använda den här taggen för att specifiera vart ordet kan brytas ifall det skulle vara för långt för att visas. Ex. Jag heter Linus. Exemplet fungerar om du testar att förminska weblsäsaren, sen sammansatta meningen kommer då delta sig där det borde vara mellanslag.
  • time/datetime - time anger precis som man enkelt kunnat gissa tid. Ex. <time>10:00</time>. Datetime kan användas för att exempelvis ange ett datum som inte specifieras i klartext. Kan se ut på följande vis: <time datetime="2017-10-13">Dagens datum</time>. Går att utveckla hela vägen ner i sekunder om så önskas. Ex. <time datetime="YYYY-MM-DDThh:mm:ssTZD">
