• mark - Med den här taggen kan man markera text. Ex. Jag heter Victor .
  • wbr - Om man har ett långt ord kan man använda den här taggen för att specifiera vart ordet kan brytas ifall det skulle vara för långt för att visas. Ex. Jag heter victor. Exemplet fungerar om du testar att förminska weblsäsaren, sen sammansatta meningen kommer då delta sig där det borde vara mellanslag.
  • time/datetime - time används för att ange tid. Ex. <time>10:00</time>. Man kan använda attributet datetime ifall man skriver en dag där datumet inte specifieras i klartext. Ex. <time datetime="2013-12-24">Jul</time>. Man kan gå så långt som sekunder och skriva vilken tidszon det gäller. Ex. <time datetime="YYYY-MM-DDThh:mm:ssTZD">
  • hcard - Används för att märka upp kontaktinformation. Ex.
    <div class="vcard">
    <span class="fn">Victor Förnemark</span>
    <span class="tel">07070707</span>
    </div>
  • hcalendar - Används för att märka upp händelser.
    <div class="vevent">
    <h2 class="summary">Web Developer Clam Bake</h2>
    <p>I'm hosting a party!</p>
    <p>It's
    <span class="dtstart" title="2011-10-25T13:30">Tuesday, October 25,
    1:30PM</span>
    at the <span class="location">Deep Sea Hotel, San Francisco, CA</span></p>
    </div>
